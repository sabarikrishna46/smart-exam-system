# 🎓 Smart Exam Management System

A full-stack web application designed to simplify and automate the examination process for educational institutions. The system enables administrators, faculty, and students to manage exams efficiently through a secure and user-friendly interface.

## ✨ Features

- 🔐 Secure Admin Login
- 👨‍🎓 Student Management
- 👨‍🏫 Faculty Management
- 📚 Subject & Department Management
- 📝 Exam Scheduling
- 🪑 Automatic Seating Arrangement
- 🎫 Hall Ticket Generation
- 📢 Notifications
- 📊 Dashboard with Key Statistics
- 📅 Attendance Management
- 📱 Responsive User Interface

## 🛠️ Tech Stack

- **Backend:** Python, Flask
- **Frontend:** HTML, CSS, JavaScript, Bootstrap
- **Database:** MySQL
- **Version Control:** Git & GitHub

## 🚀 Installation

```bash
git clone https://github.com/sabarikrishna46/smart-exam-system.git
cd smart-exam-system
pip install -r requirements.txt
python app.py
```
